A Pebble watchface written in C for Pebble, Pebble Time, Pebble Time Round, Pebble Time 2, Pebble Time Steel 2, and Pebble Core 2 Duo.

The font found in this watchface is Retro Gaming, by fontget.com user Daymarius, available for free for personal and commercial use. Find the font hosted at this link:  
https://www.fontget.com/font/retro-gaming/

The character sprite is the main character from [Shards of Grandeur](https://github.com/123outerme/Shards-of-Grandeur), my game written in Godot.
